# kudrovo-transport
Файлы, использованные в ходе выполнения курсовой работы по теме «Моделирование маршрутов общественного транспорта крупнейших городских агломераций с помощью методов машинного обучения (на примере пригородов Санкт-Петербурга)»

**podyezd_dist_to_metro.csv** - таблица с пешими расстояниями от каждого подъезда до метро

**od_matrix_top3.csv** - таблица с 3 ближайшими остановками для каждого подъезда и логит-весами

**od_matrix_top10.csv** - таблица с 10 ближайшими остановками для каждого подъезда и логит-весами

**houses_inKudrovo_pop.csv** - таблица с расчитанным населением Кудрово по домам (расчитанное - Население_округл) 

**bus_stops_v2.gpkg** - слой geopackage с автобусными остановками в Кудрово

**kud_roads_buses_v3.gpkg** - слой geopackage с УДС, пригодной для автобуса

**kud_exist_routes_v1406.gpkg** - слой geopachage с существующей маршрутной сетью Кудрово (в одну сторону)

**paradniye_with_pop_v2.gpkg** - слой парадных с населением

**baseline_routes_v2.gpkg**  - слой с построенными подходом 1 маршрутами

**ga_routes_v2.gpkg**  - слой с построенными подходом 2 маршрутами

**sue_routes_v2.gpkg**  - слой с построенными подходом 3 маршрутами

**gat_routes_v2.gpkg**  - слой с построенными подходом 4 маршрутами

**exp4_gnn_top10_routes_v2.gpkg**  - слой с построенными подходом 5 маршрутами

**baseline_kudrovo.ipynb** - ноутбук с применением подхода 1

**ga_kudrovo.ipynb** - ноутбук с применением подхода 2

**sue_kudrovo.ipynb** - ноутбук с применением подхода 3

**exp3_gat_kudrovo.ipynb** - ноутбук с применением подхода 4

**exp4_gnn_top10.ipynb** - ноутбук с применением подхода 5
